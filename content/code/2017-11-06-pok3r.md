Quelques notes mystérieuses pour configurer un clavier mécanique
/pok3r/ avec les flèches et les accents français:

```
setxkbmap -layout us -variant intl
```

```
Mode 3
AltGr + WinLeft + Space
```

```
CAPSLOCK Esc + Space = `
CAPSLOCK Esc + e     = è
CAPSLOCK Esc + a     = à
Shift Esc + Space    = ~
AltGr + ,            = ç
Shift ^ e            = ê
' + e                = é
' + Space            = '
Fn                   = ←
RightCtrl            = →
Pn                   = ↓
RightShift           = ↑
```
