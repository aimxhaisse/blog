`Docker RSS` is a Dockerfile for a tiny web RSS reader ; it can be
quickly setup in this way:

    $EDITOR docker-compose.yml
    docker-compose build
    docker-compose up -d

More on [github](https://github.com/aimxhaisse/docker-rss).
